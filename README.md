# Student Registration System

A simple **Student Registration System** developed using **Java Swing**. The application provides a graphical user interface for registering students, selecting courses and subjects, viewing registered records, deleting entries, and clearing the registration list.

---

## Overview

This project demonstrates the fundamentals of Java Swing GUI programming by implementing a student registration application. It provides an easy-to-use interface for managing student registration records using standard Swing components.

The project is suitable for beginners learning Java GUI development, event handling, and basic data management using collections.

---

## Features

- Student registration
- Course selection using radio buttons
- Dynamic subject selection based on selected course
- View registered students
- Delete selected registration
- Clear all registrations
- Exit application
- User-friendly graphical interface
- Scrollable registration list

## Functionalities

### Register Student

- Enter student name.
- Select a course.
- Choose a subject.
- Click **Registration**.
- The student's information is added to the registration list.

---

### Delete Registration

- Select a registration from the list.
- Click **Delete**.
- The selected record is removed.

---

### Clear All Records

- Click **Clear**.
- Confirm the action.
- All registration records are deleted.

---

### Exit Application

- Click **Exit** to close the application.
